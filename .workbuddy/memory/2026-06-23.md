# 2026-06-23 ZPP011 UI 修复记录

## 问题
用户报告 ZPP011 PySide6 版本界面"乌漆嘛黑"，无法正常显示和使用。

## 根本原因
在 UI 重构过程中（组件化改造），多个属性和初始化顺序出错：
1. `AlertMonitor` 初始化时缺少必需的 `data_source_func` 参数
2. `factory_combo` 引用错误（应是 `self.title_bar.factory_combo`）
3. 操作栏缺少日期范围和物料搜索控件
4. `_toggle_theme` 方法实现错误

## 修复内容

### 1. AlertMonitor 初始化 (main_window.py 第95-104行)
- 添加了 `data_source_func` 参数
- 调整了初始化顺序（`view_model` 在 `alert_monitor` 之前）

### 2. factory_combo 引用 (main_window.py 第447-454行)
- 改为 `self.title_bar.factory_combo`

### 3. 操作栏优化 (main_window.py 第246-282行)
- 添加了日期范围输入控件（`start_date_edit`, `end_date_edit`）
- 添加了物料搜索控件（`material_search_edit`）
- 添加了筛选面板切换按钮（`☰ 隐藏筛选`）

### 4. 主题切换修复 (title_bar.py + main_window.py)
- 按钮文字修正：暗色→显示"☀️ 亮色"，亮色→显示"🌙 暗色"
- 使用 `_is_dark_theme` 成员变量跟踪状态

### 5. 其他 bug 修复
- `_countdown_timer` 的 None 检查（`hasattr` → `is None`）
- `BottomBarComponent.log_text` 属性不存在（已由组件内部设置到主窗口）
- 移除 `fullscreen_btn` UI 引用（全屏仅 F11 快捷键触发）

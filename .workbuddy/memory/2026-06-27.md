# 2026-06-27 ZPP011 UI 修复记录

## 选中合计功能
裴哥选中表格行后，底部合计栏会动态显示选中行的数值合计。

### 修改内容
- `main_window.py`:
  - 重写 `_on_selection_changed`：检测选中行号，按列累加选中行的数值
  - 新增 `_update_all_summary`：取消选中时恢复整体合计
  - 重写 `_update_selection_summary`：将选中合计显示到底部栏 + 状态栏
  - `_clear_selection_summary`：无有效数值时恢复空值显示
  - `_toggle_table_fullscreen`：全屏时保留底部合计栏（不隐藏）
- `components\main_table.py`：统一合计栏标签名（配额/实际/偏差金额/偏差量）
- `components\left_panel.py`：修复输入文件显示问题（placeholder + 扩展策略）

### 行为
1. **选中行时**：底部栏显示选中行的配额/实际/偏差金额/偏差量合计
2. **取消选中**：底部栏恢复当前整体数据的合计
3. **全屏模式**：底部合计栏可见，左侧面板/日志/状态栏隐藏
4. **无有效数值**：显示 0.00

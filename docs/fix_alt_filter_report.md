## 一、问题描述

**现象**：侧边栏筛选面板中选择"替代料 = 是"后，表格未过滤出替代料行。

**根本原因**：
1. 原代码只检查 `_is_alt` 和 `是否替代料` 两个固定列名
2. 未处理布尔型和字符串类型的差异
3. 未支持多种可能的列名变体

---

## 二、修复方案

### 2.1 修改文件

`gui/app.py` - `_on_sidebar_filter_changed` 方法（第 383-407 行）

### 2.2 修复前代码

```python
# 7. 替代料筛选
is_alt = filters.get('is_alt')
if is_alt and is_alt != '全部':
    if '_is_alt' in df.columns:
        if is_alt == '是':
            df = df[df['_is_alt'] == True]
        elif is_alt == '否':
            df = df[df['_is_alt'] == False]
    elif '是否替代料' in df.columns:
        if is_alt == '是':
            df = df[df['是否替代料'] == '是']
        elif is_alt == '否':
            df = df[df['是否替代料'] == '否']
```

**问题**：
- ❌ 只支持 2 个固定列名
- ❌ 未处理数据类型差异
- ❌ 列名查找顺序固定

### 2.3 修复后代码

```python
# 7. 替代料筛选
is_alt = filters.get('is_alt')
if is_alt and is_alt != '全部':
    # 自动查找替代料列（支持多种列名）
    alt_col = None
    for col in ['_is_alt', '是否替代料', '替代料', 'is_alt']:
        if col in df.columns:
            alt_col = col
            break
    
    if alt_col:
        print(f" [DEBUG] 替代料筛选：列名={alt_col}, 筛选值={is_alt}")
        if is_alt == '是':
            # 布尔型列（如 _is_alt）
            if df[alt_col].dtype == bool:
                df = df[df[alt_col] == True]
            # 字符串列（如 是否替代料、替代料）
            else:
                df = df[df[alt_col].astype(str) == '是']
        elif is_alt == '否':
            # 布尔型列
            if df[alt_col].dtype == bool:
                df = df[df[alt_col] == False]
            # 字符串列
            else:
                df = df[df[alt_col].astype(str) != '是']
        print(f" [DEBUG] 替代料筛选后：{len(df)} 条")
    else:
        print(f" [WARN] 未找到替代料列，可用列名：{list(df.columns)}")
```

**改进**：
- ✅ 支持 4 种列名变体（`_is_alt`、`是否替代料`、`替代料`、`is_alt`）
- ✅ 自动检测数据类型（布尔型/字符串）
- ✅ 添加调试打印，便于排查问题
- ✅ 列名查找顺序灵活

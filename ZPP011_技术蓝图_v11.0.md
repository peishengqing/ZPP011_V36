### 4.3 架构整合

#### 增强排序代理模型
**文件：** `gui_pyside6/models/enhanced_sort_proxy_model.py`

**功能特性：**
- ✅ 每列独立三态循环（未排序 → 升序 → 降序 → 未排序）
- ✅ 多列联动排序（优先级栈）
- ✅ 右键菜单精细控制（单独清除某列排序）
- ✅ 排序状态可视化（箭头 + 优先级数字标签）

**技术实现：**
```python
class ColumnSortState(Enum):
    UNSORTED = 0
    ASCENDING = 1
    DESCENDING = 2

class EnhancedSortProxyModel(QSortFilterProxyModel):
    def toggle_column_sort(self, column: int) -> ColumnSortState:
        """切换指定列的排序状态"""

    def get_active_sort_rules(self) -> List[Tuple[int, Qt.SortOrder]]:
        """获取当前活跃的排序规则"""
```

**已知限制：**
- Qt 的 `QHeaderView.setSortIndicator()` 只能显示一个列的箭头
- **当前方案：**
  - 主排序列（优先级最高）显示 Qt 原生箭头
  - 其他排序列显示优先级数字标签（绿色圆圈）
- **替代方案：** （如需要可以改）
  1. 只显示主排序列箭头（简化）
  2. 自定义绘制箭头（复杂，需子类化 QHeaderView）

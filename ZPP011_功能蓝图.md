# ZPP011 生产偏差分析系统 — 功能蓝图

## 📋 项目概述

一套基于 Python PySide6 的生产偏差数据分析工具，对接 SAP 导出数据，提供**快速分析、智能审核、多维度筛选、报告生成**的一站式解决方案。

## ✅ 特点

### 优势
1. **全流程覆盖** — 从数据导入 → 分析 → AI审核 → 筛选 → 报告导出，一站式完成
2. **筛选条件丰富** — 12 种筛选项覆盖工厂/物料/偏差/状态/日期等维度
3. **替代料支持完善** — 并查集物料组（支持一对多/多对多）、净偏差自动抵消
4. **交互友好** — 右键菜单、批量操作、实时筛选、选中合计、多列联动排序
5. **数据安全** — 本地运行，无需联网（除 AI 审核外），数据不出本地
6. **性能优化** — QSortFilterProxyModel 代理模型，大表不卡顿

### 技术亮点
- **复用代码**: FilterEngine、DataFrameModel 等模块可独立复用
- **配置驱动**: 规则引擎 + 黄金模板配置，业务变化不改代码
- **容错设计**: 数值列保护、编码转化、异常日志追踪

## ⚠️ 缺点 & 待改进

### 已知不足
1. **Python 打包部署** — 当前需 Python 环境运行，未打包为独立 exe
2. **PDF 报告缺失** — 暂无 PDF 导出功能（仅支持 PPT/Excel）
3. **多用户协作** — 本地存储，无后端/数据库，不能多人共享审核状态
4. **AI 依赖网络** — AI 审核需要云端 API，离线不可用
5. **国际化** — 仅支持中文界面

### 可优化的方向
| 方向 | 建议 |
|------|------|
| 部署 | 使用 PyInstaller 打包为独立 exe，降低使用门槛 |
| 自动化 | 支持定时任务 + 邮件推送（定期自动分析） |
| 可视化 | 增加图表仪表盘（趋势图、偏差分布图） |
| 数据库 | 迁移到 SQLite/PostgreSQL，支持历史追溯 |
| 权限管理 | 多角色权限（管理员/审核员/查看者） |
| Web 版 | 使用 FastAPI + Vue 重构为 Web 应用 |

## 🐛 修复问题

### 1. 偏差数量符号反转
- **问题**：偏差数量显示反了（实际-定额 变成了 定额-实际）
- **原因**：上次修改时加了多余的反转代码
- **修复**：删除 `df['偏差数量'] = -df['偏差数量']` 这行代码

### 2. 备注列为空
- **问题**：导出Excel后，备注列显示为空
- **原因**：`sheet5_full.py` 只从"备注原因"取数，原始Excel的"备注"列没有带过来
- **修复**：改为优先取"备注"列，没有再取"备注原因"

### 3. 侧边栏筛选无效
- **问题**：侧边栏的日期筛选、物料搜索等功能无效
- **原因**：`data_frame_model.py` 第195-196行有语法错误，`return False` 缩进错位
- **修复**：修正缩进，加上缺失的 `if row_date > end_date:` 判断

### 4. 表格行号不连续
- **问题**：排序/筛选后，行号显示不连续（如 1, 3, 5...）
- **原因**：行号显示的是原始数据的索引，不是顺序行号
- **修复**：重写 `AuditProxyModel.headerData()` 方法，让行号始终按顺序显示 1, 2, 3...

### 5. 净偏差计算不正确
- **问题**：净偏差sheet的计算结果不正确
- **原因**：`net_offset.py` 有重复代码，偏差金额获取逻辑错误
- **修复**：
  - 删除第57-60行的重复代码
  - 优先取"偏差金额(含税)"列，其次取"偏差金额"列
  - 正确处理NaN值

## 📦 技术改进

### 1. DataFrame直接返回模式
- `do_analysis_v2()` 新增 `return_dataframe=True` 参数
- 分析后直接返回DataFrame，不自动保存文件
- 提高灵活性，方便后续处理

### 2. openpyxl单元格颜色填充
- 使用 `PatternFill` 为Excel单元格填充颜色
- 支持红/黄/绿三种预警等级
- 导出Excel时自动应用颜色

### 3. PyInstaller打包优化
- 使用 `--collect-all=PySide6` 自动收集所有PySide6依赖
- 避免打包后运行报错
- 打包文件大小约 248 MB

## 📋 使用说明

### 安装与运行
1. 下载 `ZPP011偏差分析器_v42.0.exe`
2. 双击运行（无需安装Python环境）
3. 如果提示"Windows保护您的电脑"，点击"仍要运行"

### 主要功能
1. **偏差分析**：选择SAP Excel文件，点击"开始分析"
2. **AI审核**：菜单栏 → 审核 → AI 审核（Mock模式，无需配置）
3. **导出Excel**：点击"导出完整Excel"，生成多Sheet报告
4. **预警查看**：表格中预警列显示颜色，汇总统计sheet也带颜色

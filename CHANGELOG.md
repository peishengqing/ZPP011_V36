## v42.8 — AI 批量审核 + 审核结果持久化 + 明细卡片

**发布日期：2026-06-11**

### 🔧 问题修复
- 修复审核结果列名中英文混乱导致重复列/数据丢失
- 修复 _替代料组 与 替代料组 同时存在的空列问题
- 修复明细卡片列名模糊匹配（空格/别名）
- 修复 AI 审核进度卡在 0% 的问题
- 修复批量 API 超时后逐条重试浪费时间（改为直接 Mock 降级）
- 修复 config/constants.py 和 __init__.py 的 VERSION 引用残留
- 修复 workers.py try/except 缩进丢失

### ⚡ 优化

- 明细卡片 _mk_label 统一创建可选中文案标签
- filter_panel 审核状态→审核结果，选项匹配 AI 输出值

### 📌 备注

## v42.7 — Agnes AI 真模型接入 + 已读持久化 + 源码备份

**发布日期：2026-06-11**

## v42.1 — PySide6 界面优化与功能完善

**发布日期：2026-06-05**

### ✨ 新增功能

#### 1. 自定义标题栏
- 浅蓝色标题栏（#1976d2），替代原生 Windows 标题栏
- 左侧显示"制作人：裴盛清"
- 右侧窗口控制按钮（最小化/最大化/关闭）

#### 2. 替代料配对区域优化
- 表格紧凑显示三列（物料A ↔ 物料B）
- 每项显示"编码|名称"，鼠标悬停显示完整信息（工厂、编码、名称）
- 新增"🔍 放大"按钮，弹出独立大窗口查看所有配对

#### 3. 预警列颜色标注（双端同步）
- **B：GUI 表格** — 预警列（红色预警/黄色预警/绿色预警）自动显示对应背景色
- **A：Excel 导出** — 导出完整分析报告时，"汇总统计"工作表预警列自动填充颜色

#### 4. 完整多 Sheet Excel 导出
- 导出时可选「快速单Sheet」或「完整多Sheet」
- 完整模式包含 10+ 个工作表：分析说明、汇总统计、替代料明细、无备注预警、中间地带明细、完整偏差明细、异常预警、偏差金额分析、偏差原因汇总、趋势分析
- 汇总统计表预警列自动上色

#### 5. 行号顺序显示
- 垂直表头行号始终按 1,2,3... 顺序显示，不受排序/筛选影响

### 🔧 问题修复
| # | 问题 | 原因 | 解决方案 |
|---|------|------|----------|
| 1 | 运行日志无内容 | Worker 线程缺少 log 信号 | 添加 `log = Signal(str)` 并连接日志框 |
| 2 | 偏差率字符串导致 abs() 崩溃 | 偏差率列包含非数字字符 | `pd.to_numeric(errors='coerce')` 强制转数字 |
| 3 | 替代料筛选无数据 | 列值 `True`/`False` 不匹配 | 标准化为"是"/"否"字符串 |
| 4 | 侧边栏日期筛选无效 | 缺少 `if row_date > end_date:` 判断 | 补充条件判断 |
| 5 | 备注列为空 | 只从"备注原因"列取数 | 优先取原始"备注"列 |
| 6 | 净偏差计算异常 | `net_offset.py` 偏差金额获取逻辑不健壮 | 优化读取顺序，兼容多列名 |
| 7 | 偏差数量符号反了 | 多余的反转代码 | 删除 `df['偏差数量'] = -df['偏差数量']` |

### ⚡ 架构优化
- **分析流程简化**：分析后不自动保存 Excel，直接显示在表格，速度更快
- **导出功能独立**：`_export_full_excel` 直接导出 DataFrame，不再依赖文件路径
- **筛选面板增强**：侧边栏可折叠，新增日期范围筛选
- **内存优化**：减少不必要的 DataFrame 拷贝
